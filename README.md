# ImageKit
A powerful kit for all image-related uses on Android platform. 

## How to use

### Step 1: Add it in your root build.gradle at the end of repositories.

```
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

### Step 2: Add the dependency.

```
	dependencies {
	        implementation 'com.github.JerryJin93:ImageKit:0.0.1'
	}
```
